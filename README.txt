# CodeRefine: Generative AI–Powered Code Review & Optimization Engine

## Description
CodeRefine is an AI-powered code review and optimization tool built using Python and Streamlit.  
It allows users to submit code or code images, which are analyzed using AI APIs (GPT/Groq/Hugging Face), and outputs **optimized, corrected, and reviewed code**.  
This project demonstrates **real-time code improvement, AI integration, and interactive frontend design**.

## Features
- Submit code for automatic review and optimization  
- Upload code images to extract code and get AI review  
- Clean, interactive interface using Streamlit  
- Integration with GPT/Groq/Hugging Face APIs for code analysis  

## Technologies Used
- **Python**: Core programming  
- **Streamlit**: Frontend interface  
- **Groq / GPT / Hugging Face APIs**: AI-based code analysis  
- **Pandas / JSON**: Data handling (if needed)
